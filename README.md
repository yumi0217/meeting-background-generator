# Meeting Background Generator

## 概要

ZoomやGoogle Meetで使用する背景画像を作成するWebアプリです。

役職・氏名・ふりがなを入力すると、背景画像上にリアルタイムで反映され、PNG形式でダウンロードできます。

## 機能

* 役職入力
* 漢字名入力
* ひらがな名入力
* リアルタイムプレビュー
* PNGダウンロード
* 必須入力チェック
* 2560×1440サイズで出力
* レスポンシブ対応

## 使用技術

* HTML
* CSS
* JavaScript
* Canvas API

## ファイル構成

```text
meeting-background-generator/
│
├─ index.html
├─ css/
│   └─ style.css
├─ js/
│   └─ script.js
└─ images/
    ├─ テンプレート.png
    └─ ダウンロードマーク.png
```

## 使い方

1. 役職を入力
2. 漢字名を入力
3. ひらがな名を入力
4. プレビューを確認
5. 「PNGダウンロード」ボタンをクリック
6. PNG画像を保存

## バリデーション

以下の項目は必須入力です。

* 役職
* 漢字名
* ひらがな名

未入力の場合はエラーメッセージを表示し、ダウンロードを行いません。

## 出力仕様

* ファイル形式：PNG
* 解像度：2560 × 1440 px
